## Description

Api to study nest with fastify

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
```

## USER ENDPOINTS

### Create a new user
endpoint: http://localhost:3000/user
method: POST
request body


- properties : {

    - name : string 
    - email: string     
}


### update user
endpoint: http://localhost:3000/user/:id
method: PUT
request body
request params


- properties : {

    - name : string 
    - email: string     
}



### delete user
endpoint: http://localhost:3000/user/:id
method: DELETE
request params


### get all users
endpoint: http://localhost:3000/user/
method: GET
